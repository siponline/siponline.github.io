# siponline.github.io

Personal academic homepage of Shi Yipeng (石一鹏), Ph.D. student at Shanghai Jiao Tong University.

## Deploy

Push to the `main` branch of `siponline/siponline.github.io`. GitHub Pages will serve it automatically at `https://siponline.github.io`.

## Local Preview

```bash
python -m http.server 8080
# or simply open index.html in your browser
```
